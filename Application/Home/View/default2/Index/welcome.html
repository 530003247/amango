一站式

Amango 采用业内先进的 CMS 理念，基于Onethink框架进行二次开发，从CMS逐步扩展到微信公众号管理，最终实现所有资源的统一调配，适应大数据入口潮流。

模块化

Amango 除了支持Onethink插件的开发模式、也支持独有的微信插件开发，双插件机制，可快速构建界面出色、体验优秀的跨屏微信公众管理系统，大幅度提升你营销效率。

本地化

Amango紧紧围绕关键词展开，关键词可调用本站所有资源、接口、插件，为了公众号的个性化，我们提供了多种特色性的微信机制玩法。充分利用平台良好的拓展性。

高性能

Amango 非常注重性能，基于轻量的 Onethink 开发，并使用 Amaze 来做前台css框架，让你的 微信 + Web 应用可以高速载入。
